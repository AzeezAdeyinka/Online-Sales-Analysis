### Project Overview
This project focuses on analyzing company sales online. 
The goal was to explore Total Sales reached, Total Orders, Sales Trends, Sales by payment method and so on.
The insights were visualized using an interactive dashboard built in Power BI to help stakeholders better understand the current sales of the company.

### Data Source
Kaggle

### Tools
- Power BI and Dax

### Data Cleaning
- Removed missing or irrelevant data points (e.g., incomplete responses)

- Converted text-based data into numerical values for analysis (e.g., sales, orders)

- Filtered outliers to ensure realistic visual representation of sales
  

### Data Analysis
Key insights derived from the dashboard include:

-  Despite MoM declines in total orders (-3%) and customers (-7%), Average Order Value increased by 6%, signaling stronger customer spend and effective value optimization.

- European markets (Norway, Germany, Belgium, France) consistently lead in revenue contribution, reflecting strong regional demand.

- Digital payments dominate, with PayPal and Bank Transfer contributing the majority of transactions.

- A low return rate (10.5%) indicates solid product quality and customer satisfaction over time.

- Stationery and Accessories remain top-performing categories, while Royal Mail leads shipment volumes.

### Results 
- Long-term revenue growth is being driven more by customer value and purchasing behavior than sheer order volume.
